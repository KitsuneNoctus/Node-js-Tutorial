##### Install Node
- Update Homebrew first

    $ brew update

- Install Node

    $ brew install node

- Test / Check Version

    $ node -v

- Update

    $ brew update
    $ brew upgrade node

- Uninstall

    $ brew uninstall node

##### Packages

$npm i lodash

$npm  expressjs/express
